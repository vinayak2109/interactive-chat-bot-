# AI Intent-Based Chatbot

A Python-based intelligent chatbot capable of understanding natural language and classifying user intents.

## Features
- **Natural Language Processing:** Uses NLTK for tokenization and lemmatization.
- **Deep Learning:** Custom Neural Network built with TensorFlow/Keras.
- **Context Awareness:** Can handle follow-up conversations.
- **Local Inference:** Supports Llama 3 via LM Studio for offline privacy.

##  Tech Stack
- Python 3.12
- TensorFlow (Keras)
- NLTK (Natural Language Toolkit)
- NumPy

##  How to Run
1. clone the repo:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/ai-chatbot-project.git](https://github.com/YOUR_USERNAME/ai-chatbot-project.git)
2. install dependancies:   
   pip install numpy nltk tensorflow
3. run the training script:   
   python train_chatbot.py
4. Start chatting:
   python chat.py
